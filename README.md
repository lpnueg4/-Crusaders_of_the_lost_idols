## How to use Pipfile

  1. `pip install pipenv==2018.5.18`
  2. `git clone https://github.com/lpnueg4/G_Crusaders_of_the_lost_idols.git` 
  1. `cd G_Crusaders_of_the_lost_idols`
  4. `pipenv install`
  5. `pipenv run python plot.py`
